
안녕하세요 2조의 발표 시작하겠습니다.

발표 순서로는 먼저 승택님과 유진님의 기능 명세서 설명과  
재성님의 DB 모델링 설명 후
저와 두수님의 화면설계 설명 순 으로 진행됩니다.

기능 명세서 설명에 앞서 저희 2조의 개발 주제를 말씀드리겠습니다.
저희 조는 프로젝트 ,일정, 고객 정보 관리를 중점으로 하는 시스템 개발을 맡게 되었습니다.
그리고 이런 기능을 사용할 많은 고객사 중 IT 회사를 타겟으로 기능 명세작업을 시작했는데요,
그 결과 다음과 같은 주요 기능들이 도출 되었습니다.

프로젝트 관리를 위한 관리 기능, 일정 관리를 위한 캘린더와 휴가, 출장 관리 기능,
고객사 관리 기능, 시스템 관리자 기능, 사이트 매니저 기능, 추가로 결재 기능과 게시판 기능, 알림 기능이 도출 되었습니다.

이제 위 기능들 중 저희 사이트의 특이 기능 몇가지를 설명 드리겠습니다.





<<피그마 흐름도>>

-- 로그인
먼저 로그인 화면에서 아이디, 비밀번호를 입력해 로그인을 하면(로그인 버튼 클릭) 가정 먼저 프로젝트 목록 화면으로 이동합니다

-- 프로젝트
프로젝트 목록은 기본적으로 카드 형식으로 표시되며 (리스트 버튼 클릭) 리스트 형식으로 전환해서 조회도 가능합니다.

아래 프로젝트 등록 버튼을 누르면(등록 버튼 클릭) 프로젝트 생성 화면으로 이동하며 여기서 프로젝트의 기본 정보를 입력하여
프로젝트 항목을 생성할 수 있습니다. (등록 버튼 클릭)

생성하면 등록한 프로젝트의 상세 페이지로 이동하게되는데 프로젝트 정보와 투입된 인력의 투입 현황을 볼 수 있습니다.
아래 첨부파일 탭을 누르면(첨부파일 탭 클릭) 첨부파일 화면으로 이동하게 되며 해당 프로젝트와 관련하여 등록된 파일들을 볼 수 있습니다.

--------------------- (두수)

지금까지 사원 계정의 기능들을 설명이였습니다. 다음으로 매니저 기능을 설명 드리겠습니다.
 
(매니저 화면 이동)
-- 매니저
매니저 계정으로 접속하면 다음과 같이 매니저만 볼 수 있는 매니저 메뉴가 존재합니다.

-- 매니저 휴가관리
그 중 휴가관리를 클릭하면(휴가 관리 클릭) 전체 사원들의 목록이 조회됩니다.

그 중 수정이 필요한 사원의 수정 버튼을 클릭하면 (수정 버튼 클릭) 해당 사원의 휴가 정보 화면으로 이동합니다.
이 화면에선 해당 사원이 잘 못 등록하였거나 필요해 의해 등록한 휴가를 취소해야 할 경우 삭제 버튼으로 취소할 수 있습니다.

-- 매니저 공지사항
다음은 공지사항 메뉴입니다. (공지사항 메뉴 클릭)

공지사항 메뉴에서 매내저 계정에게만 작성하기 버튼이 보여지며 해당 버튼 클릭 시(작성하기 클릭) 공지사항 작성 화면으로 이동하하게 되고,
내용 입력 후 작성하기 버튼을 클릭해(작성하기 버튼 클릭) 공지사항을 등록할 수 있습니다.

(첨부파일 밑에 빈 하얀 공간 클릭)

-- 시스템 관리자
마지막으로 시스템 관리자 기능입니다.

시스템 관리자 계정으로 접속 시 일반 사원, 매니저 계정과는 다르게 접속할 수 있는 메뉴가 다르게 표시됩니다.

-- 시스템 관리자 계정 관리
그 중 계정관리 메뉴의 계정 목록 화면에선 사이트에 등록 된 모든 계정을 볼 수 있으면 필터링과 정렬, 검색 기능으로 원하는 방식으로 조회가 가능합니다.

상단의 생성 버튼을 클릭하면 (계정 생성 클릭) 계정 생성 화면으로 이동하게 되며 개인 신상 정보와 사내 정보를 입력하여 (계정 생성 클릭) 신규 계정을 생성할 수 있습니다.

-- 시스템 관리자 환경 설
마지막으로 환경 설정 메뉴를 클릭해보면(환경 설정 메뉴 클릭)  환경설정 조회 화면으로 이동하게 됩니다.

이 화면에서 회사 정책에 맞게 각 기능들에 필요한 항목 값들을 관리할 수 있습니다.

항목 종류 옆 버튼을 클릭하면 ( + 버튼 클릭) 추가 모달이 뜨게 되며 새로운 항목을 추가할 수 있습니다.(빈 곳 클릭)

기존 항목의 수정이 필요하면 수정버튼을 클릭하여(연차 옆 수정 버튼 클릭) 올라온 수정 모달창에서 해당 항목의 값을 변경할 수 있습니다.
그리고 항목의 개수가 많아져서 화면에 다 표시되지 않을 경우 스크롤바가 생성되어 리스트를 내리고 올려서 확인할 수 있습니다.
